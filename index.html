3D Motion Forecasting

MolmoMotion forecasts an object's future 3D motion from a language instruction, across diverse objects and scenes. Select an example.

Downstream: Robotics Planning

The 3D motion prior transfers across embodiments. After fine-tuning on DROID, MolmoMotion predicts coherent object trajectories on real robots.

Downstream: Video Generation

MolmoMotion's predicted 3D trajectories guide image-to-video models to follow the requested action more faithfully. Select a prompt to compare methods.

The MolmoMotion Model

MolmoMotion uses Molmo 2 as its vision-language backbone and predicts each query point's future 3D trajectory in two variants: an autoregressive model (MolmoMotion-AR) that decodes coordinates as text, and a flow-matching model (MolmoMotion-FM) that generates trajectories in continuous 3D space.

MolmoMotion-1M and PointMotionBench

An automatic annotation pipeline extracts object-grounded 3D point trajectories from unconstrained video, yielding MolmoMotion-1M — the largest such corpus to date, spanning 736 motion types and 5,692 objects from 1.16M clips.

PointMotionBench is a human-verified benchmark of 2,720 held-out clips across 111 object categories and 61 motion types.

Limitations

MolmoMotion uses only 8 query points per object, which limits dense geometry and complex deformable motion; broader real-world and closed-loop evaluation remains future work.
